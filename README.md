# Healthcare Analytics System

A Healthcare Analytics System developed using **MySQL**, **XAMPP**, and **Python**. This project demonstrates database design, SQL operations, and backend integration for managing healthcare data efficiently.

## Features
- Patient Management
- Doctor Management
- Hospital Management
- Visit Records
- Laboratory Test Records
- Prescription Management
- Billing Information
- CRUD Operations (Create, Read, Update, Delete)

## Technologies Used
- Python
- MySQL
- XAMPP (Apache & MySQL)
- MySQL Connector for Python



## Requirements
- Python 3.x
- XAMPP
- MySQL Server
- mysql-connector-python

Install the required package:

```bash
pip install mysql-connector-python
```

## How to Run
1. Start Apache and MySQL using XAMPP.
2. Import the SQL database into MySQL.
3. Update the database credentials in the Python file.
4. Run the backend application.
